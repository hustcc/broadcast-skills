# Broadcast Skills

> Batch send operational messages to IM groups.

## Overview

Broadcast Skills provides automated message broadcasting capabilities for popular IM platforms. It enables you to quickly push feature updates, community announcements, and operational content to multiple groups in one go.

> [!WARNING]
> This project only merges AI-generated code.
>
> **How to contribute:**
> 1. Submit an issue describing the problem clearly
> 2. Assign it to @copilot with your requirements

## Usage

Add this marketplace to Claude Code:
```bash
/plugin marketplace add hustcc/broadcast-skills
```

Or you can directly install the skills for your multiple agents:

```bash
npx skills add hustcc/broadcast-skills
```

## Available Skills

- 📢 **dingtalk-broadcast**: Batch send operational messages to DingTalk groups. Supports text, markdown, link, and actionCard message types.

- 💬 **wecom-broadcast**: Batch send operational messages to WeCom (企业微信) groups. Supports text, markdown, and news message types.
